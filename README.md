# 🚀 SmartRes AI: Code the Future Edition

### 🔗 [Live Hosted App Link] | 🎥 [Loom Demo Video Link]

## 📌 Project Overview

**SmartRes AI** is an intelligent resume analysis platform built for the **UnsaidTalks: Code the Future Hackathon**. Our mission is to bridge the gap between talented candidates and recruiters who spend only seconds scanning a document.

Most resumes fail due to unclear formatting or missing key skills. This tool provides instant, AI-driven feedback to help students and professionals achieve a high-impact resume score of at least **8/10**.

---

## ✨ Key Features

* 
**Instant Extraction:** Automatically identifies core skills, key projects, and industry keywords from uploaded PDFs.


* 
**AI Scoring Engine:** Provides a quantitative rating out of 10 based on clarity, professional tone, and impact.


* 
**JD Alignment & Gap Analysis:** Compares the resume against a target Job Description (JD) to identify missing keywords and alignment gaps.


* 
**Summary Generator:** Crafts a concise, 2-line professional summary designed to capture recruiter attention.


* 
**Actionable Roadmap:** Gives specific suggestions for every section, including a task to re-evaluate and improve the score.



---

## 🛠️ Tech Stack

* 
**Frontend & UI:** Streamlit (chosen for rapid prototyping and clean UX).


* 
**LLM Engine:** Google Gemini 1.5 Flash (evaluated against leading models like OpenAI and Claude for quality).


* **PDF Processing:** PyMuPDF / PyPDF2.
* **Language:** Python.

## 🚀 Getting Started

### Prerequisites

* Python 3.9+
* A Gemini API Key (or OpenAI/Claude)

### Installation

1. **Clone the Repository:**
```bash
git clone https://github.com/RIP09/AI_Resume_Analysis.git
cd your-repo-name

```


2. **Install Dependencies:**
```bash
pip install -r requirements.txt

```


3. **Set Up Environment Variables:**
Create a `.env` file and add your API key:
```text
GOOGLE_API_KEY=your_api_key_here

```


4. **Run the Application:**
```bash
streamlit run app.py

```
